# Tetris Game

### Setup environment
> ### macos
```
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
```

> ### window
```
- python -m venv venv
- venv\Scripts\activate
- pip install -r requirements.txt
```

### Run application
```
python app.py
```
